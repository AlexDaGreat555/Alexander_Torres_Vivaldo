Super Smash Bros Card Game

I developed a Super Smash Bros-themed memory card matching game using HTML, CSS, and JavaScript. The game includes interactive cards that flip when clicked revealing the characters while tracking the number of flips. I used HTML for structure, CSS for layout and animations, and JavaScript to handle the game logic, including card flipping, matching, and score updates. I also implemented CSS 3D transformations, state management, event listeners, and a shuffle function using an Immediately Invoked Function Expression (IIFE) to randomize card order.

Website Link: https://alexdagreat555.github.io/ssb_cardgame/

Github Link: https://github.com/AlexDaGreat555/ssb_cardgame

Microsoft Paints

In my Microsoft Paints project, I developed a user-friendly prototype using Python that allows users to draw various shapes like squares, circles, straight lines, and polygons. The program also includes a free-write tool for sketching and a fun “Pou” feature inspired by the character from the popular mobile game. I built the project using CMUs CMU_Graphics library, which streamlined the graphical elements.

The program is structured using top-down design, breaking down complex features into manageable tasks, and follows object-oriented programming (OOP) principles, ensuring each drawing tool is modular and easily maintainable. By applying OOP, I organized the code to be reusable and scalable, making it easy to extend with new features in the future.

Github Link: https://github.com/AlexDaGreat555/Microsoft_Paints_Py

Battleship

In my Battleship project, I programmed a very basic game of battleship using python. The game simulates a classic Battleship setup, where both the player and the bot place their ships on 10x10 grids, and then take alternating turns to try to hit each others ships. The player interacts by clicking to place their ships and make guesses, while the bot randomly selects cells to target on the players grid. The game concludes when one players ships are all hit or the maximum number of turns is reached, declaring a winner or a draw.

Github Link: https://github.com/AlexDaGreat555/basic-battleship-py
